# Introduction

The codebase of the WallR Android App.

## Components
 <ul>
  <li> Clean Architecture with MVP </li>
  <li> Dependency injection using Dagger </li>
  <li> Multi-threading using RxJava2 </li>
  <li> Unsplash Api </li>
  <li> Firebase Realtime Database </li>
  <li> Use of Room database and Shared preferences for storing information locally </li>
  <li> JUnit and Mockito for unit testing </li>
 </ul>
